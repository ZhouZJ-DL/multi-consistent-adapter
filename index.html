MC-Adapter: Multi Turn Consistent Image Editing

Zijun Zhou*,
Institute of Automation, Chinese Academy of Sciences
*Indicates Equal Contribution

Overview

One-shot prompting has clear limitations—it's often difficult for users to fully express their needs in a single attempt, and generating an image that satisfies them on the first try can be equally challenging. This makes it natural to consider creating an interactive multi-turn editing framework. Such a framework would let users refine their requests step-by-step, gradually arriving at a result they're happy with. This system would also allow users to base their edits on previously generated images. The resulting image sequence should retain a consistent visual style and enable fine-grained adjustments at each step, helping users refine the image in a specific direction.

Multi-Turn Edit Pipeline

A potential project could involve storing image embeddings at each generation round. When initiating a new editing round, historical embeddings can be merged, with higher weight assigned to the user-selected base image, followed by implementing edits guided by the new text prompt. Further experiments might include editing the initial noise or merging attention maps. Careful consideration of merging strategies will be essential to achieve accurate and reliable information integration.

Display Demo

In the demo, users start by uploading one or more images. They select a base image and enter a text prompt to begin editing. After each editing round, users can choose a previously generated image from the history as the base image to continue the editing process.
